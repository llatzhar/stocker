# ドローダウン検出ツール

## 概要

投資信託の基準価額CSVを読み込み、最高値からの3%刻みドローダウン発生をリストアップするコマンドラインツール。

## 使い方

```bash
python drawdown.py 253425.csv
```

## 入力仕様

- **ファイル形式:** CSV（Shift_JIS エンコーディング）
- **ヘッダー:** 2行（スキップ）
- **使用列:**
  - 1列目: 日付（YYYY/MM/DD）
  - 2列目: 基準価額

### CSVサンプル

```
（ヘッダー1行目）
（ヘッダー2行目）
2018/10/31,10000,10000,,0.10
2018/11/01,10077,10077,,0.10
```

## 検出ロジック

1. CSVを先頭行（古い日付）から順に走査する
2. 基準価額の過去最高値を追跡する
3. 現在値が最高値から **3%の倍数** だけ下落した時点で検出・報告する
   - 閾値: 3%, 6%, 9%, 12%, 15%, ... （3%刻み）
4. 同一の下落局面では段階的にすべて報告する
   - 例: 9%下落した場合 → 3%, 6%, 9% の3回報告
5. 最高値が更新されたら、次の検出閾値を **3%にリセット** する

## 出力仕様

標準出力に1行1イベントで出力する。

### フォーマット

```
日付: 最高値 XXXXX → 現在値 YYYYY (-Z.Z%) [N%ドローダウン突破]
```

### 出力例

```
2020/03/16: 最高値 12000 → 現在値 11400 (-5.0%) [3%ドローダウン突破]
2020/03/18: 最高値 12000 → 現在値 11100 (-7.5%) [6%ドローダウン突破]
```

## 制約・前提

- 基準価額が空欄・不正値の行はスキップする
- 日付は昇順（古い→新しい）で格納されている前提
